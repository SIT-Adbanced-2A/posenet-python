# Throw Detecter
This program detects that someone has thrown an object and cuts and saves the moment

# Requirements
- python3.6
- tensorflow-gpu
- scipy
- pyyaml
- cython

# Getting Started
1. Build library
```
cd mylib
setup.py build_ext --inplace
```

2. Execute Throw Detecter
```
cd ..
python throw_detecter.py movie_file
```
